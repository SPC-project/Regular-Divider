v0.4.1
[2017-06-04]
### Добавленно:
- Иконки для типов треугольников
- Треугольник дополняется до прямоугольника воздушными элементами, с тех
сторон также можно строить воздушные элементы
- Sorter.py может отсортировать узлы по координатам и соответственно
модифицировать элементы. Его можно вызвать из GUI
- Sorter.py может отсортировать элементы по координатам. Можно вызвать из GUI
- Можно задать толщину воздушного слоя для всей фигуры целиком
- Координатная сетка (отключаема)
### Изменения:
- Удален дублирующий функционал для редактирования фигуры — остались
вызываемые через контекстное меню диалоги
- Материал узла больше не сохраняется (отдано предпочтение материалу элемента)
- Разбиение примитивов совершается в "естественном порядке": верхний и левый
воздух, фигура, правый и нижний воздух
### Рефакторинг:
- Общий код NewRectangleWidget и NewTriangleWidget вынесен в абстрактный класс
AbstractNewWidget
- Переписаны тесты в соответствии с изменениями для треугольников; тестируется
теперь и материал элементов
- Новый алгоритм разбиения, с удалением избыточных узлов (Combiner.py). Также,
в удаляются избыточные элементы в случае наложения (у фигуры есть приоритет)
### Исправления:
- При редактировании присоединённых треугольников отключаются те варианты, что
могли бы разорвать сетку


v0.4.0
[2016-12-26]
### Добавлено:
- Второй тип примитивов, треугольник, с соответствующим UI
- Два новых примера фигур в examples/
- Подсказка с текущим размером холстов при изменении размеров окна
### Изменения:
- Новый формат для файлов .d42do (фигуры в examples/ обновлены)
- Часть кода вынесена из Rectangle в Abstract_primitive
- Создана обертка NewPrimitiveDialog для ввода данных для генерации примитивов
- Mesh теперь отдельный класс, что хранит параметры сетки и вспомогательную
информацию: тип примитива и (в случае треугольников) его вид (form)
### Исправления:
- Содержимое холстов (QPixmap) больше не налазит на границы рамки (QFrame)
- В диалоге генерации примитивов были недоступны стрелки QSpinBox'a левой
стороны 
- Чуть изменен размер рабочего окна чтобы холсты были по-умолчанию квадратные
и включено сглаживание (antialiasing) — что повышает качество картинки
