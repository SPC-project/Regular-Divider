v0.4.0
[2016-12-26]
### Добавлено:
- Второй тип примитивов, треугольник, с соответствующим UI
- Два новых примера фигур в examples/
- Подсказка с текущим размером холстов при изменении размеров окна
### Изменения:
- Новый формат для файлов .d42do (фигуры в examples/ обновлены)
- Часть кода вынесена из Rectangle в Abstract_primitive
- Создана обертка NewPrimitiveDialog для ввода данных для генерации примитивов
- Mesh теперь отдельный класс, что хранит параметры сетки и вспомогательную
информацию: тип примитива и (в случае треугольников) его вид (form)
### Исправления:
- Содержимое холстов (QPixmap) больше не налазит на границы рамки (QFrame)
- В диалоге генерации примитивов были недоступны стрелки QSpinBox'a левой
стороны 
- Чуть изменен размер рабочего окна чтобы холсты были по-умолчанию квадратные
и включено сглаживание (antialiasing) — что повышает качество картинки
